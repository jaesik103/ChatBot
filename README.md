# AI_chat_bot

선톡용 챗봇


# 목적
- 선톡을 하기전 예행연습 해볼수 있는 챗봇
- nlp 공부 겸 제작해 본, 토이프로젝트
- pre-training 된 모델에 새 데이터로 fine tuning 하여 새로운 챗봇 제작
- [streamlit][1] 사용

# Python version
- 3.8.9

# 사용 모델 및 소스
- [KoBART-chatbot][2]

# fine tuning 데이터
- [국립국어원 온라인 대화 말뭉치(버전 1.0)][3]를 챗봇 목적에 맞게 전처리 한 후 사용

# 사용법
- streamlit
